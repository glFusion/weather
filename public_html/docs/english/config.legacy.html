Weather Plugin

The Weather Plugin allows you to display weather information from wunderground.com or worldweatheronline.com. Included is a PHP block to show current conditions, and current or forecasted weather can be displayed in other content via autotags. Additionally, API functions are available to embed weather information into other plugins.

This version requires GLFusion 1.2.0 or higher.

Weather Plugin Configuration Options

Main Weather Settings
Option Description
Anonymous Access Select the plugin access to give anonymous visitors.
  • None - The plugin page is not available. This does not affect the visibility of the PHP block; see Locations to show in block.
  • View - Visitors may view the forecast page.
  • View & Search - Anonymous visitors may view the weather page and submit locations to search.
Display glFusion Blocks Select the glFusion blocks that will be displayed when the plugin page is displayed.
Minutes to cache weather data Enter the number of minutes to cache weather data. When data is retrieved from the provider, it is cached in a database table to reduce the number of lookups made. This number should be set to a reasonable value based on how frequently the weather is expected to change. 120 minutes (2 hours) is a good start.
Default Location Enter a location to be used as the default. If a user does not have a persional location value filled in, or if the Locations to show in block is "Default Only", this location will be used.
Show the plugin on the "Plugins" menu? Select "Yes" to have a menu option for this plugin page added to the default "Plugins" menu. If this is "No", no menu option will be added but the page will still be available if added to another menu or linked from other content.
Weather Provider Select the weather provider to use. Remember that you'll need to sign up on the provider's website and set the API key value below.

Be sure to purge the weather cache (in addition to the glFusion template cache) after changing providers.
  • World Weather Online
  • Weather Underground
Windspeed units Select the unit of measure for Windspeed- Miles per Hour or Kilometres per Hour
Temperature units Select the unit of measure used for temperature- Farenheit or Celsius
World Weather Online Settings
Option Description
API Key Enter the API key assigned to you by World Weather Online. Visit worldweatheronline.com to sign up for a key.
Weather Underground Settings
Option Description
API Key Enter the API key assigned to you by Weather Underground. Visit wunderground.com to sign up for a key.
Referral Key Enter the referral key assigned to you by Weather Underground. WU offers additional "raindrops" based on clicks received from your site.
