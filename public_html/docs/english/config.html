glFusion Weather Plugin Configuration

Main Settings

Anonymous Access
Select the access for Anonymous users. Default is view only, if you set this to View & Search be aware that you might exceed your API search quotas.
Display glFusion Blocks
Select the glFusion blocks that will be displayed when the plugin page is displayed.
Minutes to cache weather data
Enter the number of minutes to cache weather data. When data is retrieved from the provider, it is cached in a database table to reduce the number of lookups made. This number should be set to a reasonable value based on how frequently the weather is expected to change. 120 minutes (2 hours) is a good start.
Default Location
Enter a location to be used as the default. If a user does not have a persional location value filled in, or if the Locations to show in block is "Default Only", this location will be used.
Location for PHP Block
Select which location will be shown in the PHP block (if used). Either the default location (above) or the user's personal location (if available) can be shown.
Show the plugin on the "Plugins" menu?
Select "Yes" to have a menu option for this plugin page added to the default "Plugins" menu. If this is "No", no menu option will be added but the page will still be available if added to another menu or linked from other content.
Weather Provider
Select the weather provider to use. Remember that you'll need to sign up on the provider's website and set the API key
Windspeed units
Select the unit of measure for Windspeed- Miles per Hour or Kilometres per Hour
Temperature units
Select the unit of measure used for temperature- Farenheit or Celsius
API Key
Enter the API key assigned to you by World Weather Online. Visit worldweatheronline.com to sign up for a key.
API Key
Enter the API key assigned to you by Weather Underground. Visit wunderground.com to sign up for a key.
Referral Key
Enter the referral key assigned to you by Weather Underground. WU offers additional "raindrops" based on clicks received from your site.
API Key
Enter the API key assigned to you by APIXU. Visit apixu.com to sign up for a key.
